# Credit Risk Analysis - Probability of Default Model

## Objective
Build a Logistic Regression model to predict Probability of Default (PD) using customer loan data.

## Steps Performed
- Data preprocessing
- Feature selection
- Train-test split
- Logistic Regression model training
- Model evaluation (Accuracy, Confusion Matrix)
- Probability of Default estimation

## Model Performance
- Accuracy: 99.7%
- Recall (Default Detection Rate): ~98%

## Tools Used
- Python
- Pandas
- Scikit-learn
- Google Colab
